This tool helps you convert numbers between different bases: hexadecimal (HEX), decimal (DEC), octal (OCT) and binary (BIN).
Recommended usage:
  • Select the base of the number you want to input by clicking the corresponding button (HEX, DEC, OCT or BIN). The active base button is highlighted.
  • Use the on-screen buttons to enter the number. Only valid digits for the selected base will be enabled.
  • The converted values will be displayed in the respective fields for each base.
  • You can clear the input field using the CLR (clear) button or delete the last digit with the BS (backspace) button.
